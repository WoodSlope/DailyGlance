# DailyGlance

DailyGlance 是一个面向 A 股大盘与自选股观察的网页量化终端，用来集中查看指数环境、个股走势、技术指标和辅助交易结论。

> 仅用于观察、研究和复盘，不构成投资建议。

## 功能

- 大盘：上证指数、创业板指、中证 1000、科创 50
- 自选：搜索股票并加入本地自选
- 图表：K 线、成交量、MACD、KDJ
- 周期：日线 / 周线
- 区间：90 天 / 180 天 / 360 天
- 均线：MA5、MA10、MA20、MA30、MA60、MA120、MA250
- 策略：稳健趋势、波段抄底、突破追涨、综合全能
- 输出：交易结论、建议仓位、防守位、压力区、三条关键依据
- 工具：手动更新、回测、清缓存、设置、帮助、性能诊断

## 页面结构

- 左侧：大盘列表 / 自选股列表
- 中间：主图和指标图
- 右侧：价格、结论、依据、风险提示

## 使用

1. 左侧选择大盘或个股
2. 中间看图
3. 右侧先看结论，再看依据
4. 顶部切周期、区间和均线

## 结论结构

- 第 1 条：大盘定基调
- 第 2 条：指数动能 / 个股信号
- 第 3 条：风控 / 防守

结论卡优先展示最终交易动作，依据只保留必要信息。

## 数据更新规则

- 首屏：先读本地缓存，再做后台刷新
- 当前正在看的大盘或个股：盘中约每 30 秒自动刷新一次
- 大盘左侧列表：随大盘数据刷新一起更新
- 自选左侧列表：盘中、位于自选页时，约每 90 秒低频刷新一次
- 切到某只个股后：约 2 秒后会补刷其它自选股
- 手动更新：同一标的 60 秒内限制频繁点击
- 历史数据刷新有冷却：15 秒成功冷却、8 秒尝试冷却
- 分红 / 复权保护：最后一日价格偏差超过 2% 时，不直接增量拼接，改走整段覆盖

## 缓存与一致性

- 本地缓存使用浏览器 IndexedDB
- 静态资源通过 `?v=...` 版本参数加载
- 如果本地和 GitHub Pages 的样式、图表或结论不一致，优先检查：
  - 浏览器强制刷新
  - 页面右上角清除缓存
  - `index.html` 中静态资源版本号是否已更新

## 文件

- `index.html`：入口
- `assets/css/dailyglance.css`：样式
- `assets/js/01-config-ui.js`：全局配置、UI 工具
- `assets/js/02-data.js`：数据、缓存、同步
- `assets/js/03-calculations.js`：指标、信号、仓位推导
- `assets/js/04-render.js`：图表和右侧渲染
- `assets/js/05-app.js`：初始化、交互、生命周期

## 部署

上传：

```text
index.html
assets/
```

发布时注意：

- 改了 CSS 或 JS 后，建议同步更新 `index.html` 里的 `?v=...`
- 当前采用普通 `<script>` 顺序加载，5 个脚本顺序不要改

## 风险声明

所有结论、信号、仓位建议和回测结果仅用于研究、观察和复盘。工具输出不能替代独立判断，也不构成买卖建议。
